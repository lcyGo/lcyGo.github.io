农业农村部：1—4月份乡村消费品零售额同比增长10.7%

2018-05-22 16:28:00

人民网北京5月22日电 据农业农村部网站消息，农村消费、投资、出口保持较快增长，乡村发展潜力不断释放。农村居民消费结构加快升级，乡村线上线下消费环境日益改善，农村消费市场日趋活跃，增速保持在两位数。1—4月份，乡村消费品零售额达到17193亿元，同比增长10.7%，增速较城镇快1.1个百分点。

4月份以来，各级农业农村部门按照党中央、国务院关于实施乡村振兴战略的决策部署，认真贯彻落实高质量发展要求，统筹抓好春管春播、结构调整、产业融合发展等各项工作，农业农村经济继续保持平稳向好态势，呈现生产形势总体稳定、结构调整持续优化、市场供给量足价稳、产业质量效益提升的良好局面。

夏粮丰收有基础，早稻长势较好。4月以来，冬麦区几次出现大范围降雨过程，有效改善了土壤墒情，有利于小麦生长发育，长势接近常年，且病虫害总体偏轻。油菜籽产区大部光热适宜、墒情良好，有利于油菜生长发育和产量形成。早稻栽插结束，预计播种面积较上年减少500多万亩，目前大部处于分蘖至孕穗期，长势总体略好于去年。

春播进度接近八成，种植结构持续优化。春播期间，各类农机具配备充足，预计春播粮油作物机耕率稳定在90%以上、机播率达到52%。据农业农村部农情调度，截至5月17日，全国春播农作物已完成意向的79.6%，进度同比略快。种植结构进一步优化，预计水稻面积调减1000万亩以上，供给短缺的大豆面积增加，优质稻谷、高蛋白大豆、高产高糖甘蔗、优质饲草等面积也明显增加。

生猪产能继续调减，水产海洋捕捞强度下降。随着大中型养殖企业前期增加产能逐步释放，生猪供求转向宽松、价格低位运行。据农业农村部定点监测，4月份生猪、能繁母猪存栏环比分别下降0.8%和1.4%，同比分别下降1.5%和2.1%。蛋鸡产能基本调整到位，养殖行情较好。4月份产蛋鸡存栏同比下降4%，1—4月份鸡蛋产量同比下降6.4%，每只产蛋鸡累计盈利6.45元。肉牛、肉羊养殖效益稳定，存栏同比分别增加1%和2.9%。水产海洋捕捞强度下降，减量效果明显。据农业农村部统计，1—4月份国内水产品总产量同比增长1.1%，其中海洋捕捞产量同比下降2.6%。

大宗农产品价格略降，部分鲜活农产品价格跌幅较大。据国家统计局统计，4月份CPI同比上涨1.8%，环比下降0.2%；食品价格同比上涨0.7%、环比下降1.9%，为稳物价起到“压舱石”作用。其中，受政策性储备粮拍卖轮库影响，稻谷和小麦价格小幅下跌，玉米产区批发价环比回落2.7%；受大豆加工补贴政策出台及国际贸易形势影响，企业收购大豆积极性有所提升，大豆、豆油持平略涨。畜禽价格全面下降，猪肉、鸡蛋价格连续3个月下降，累计跌幅分别达18.4%和21.9%；牛羊肉进入消费淡季，集贸市场均价环比分别下降0.9%和1.1%。4月份我国四大流域进入全面禁渔期，水产品市场供给偏紧、价格同比上涨4%。蔬菜供应充足、价格下跌，优质水果供给增加、价格上涨，农业农村部重点监测的28种蔬菜、7种水果月均批发价环比分别下降7.3%、上涨2%。

农产品加工业稳中增效，新产业新业态发展强劲。一季度，规模以上农产品加工业企业实现主营业务收入40647亿元，同比增长7%，增速比去年全年上升0.5个百分点；实现利润总额2780亿元，同比增长7.9%。其中，农产品精深加工呈现快速发展态势，增速保持在8%以上，较初级加工增速快3个百分点左右。休闲农业和乡村旅游持续走热，亲子游和以采摘、赏花、乡村农事体验等为主的休闲度假游成为亮点。鲜活农产品线上销售异军突起，带动农产品电子商务快速发展。据淘宝（含天猫）大数据平台显示，1—4月份粮油、食品、饮料、烟酒等产品销售额同比增长26.4%。

各地区纷纷出台关于实施乡村振兴战略的文件，制定并落实引导社会资本进入农业农村的优惠政策，农村日益成为投资的“蓝海”。1—4月份，第一产业固定资产投资(不含农户)达到4591亿元，同比增长16.8%，较二三产业增速分别快14.3个和7.5个百分点。全球经济复苏，国际市场需求企稳，“一带一路”农业合作领域不断拓展，我国农产品出口回暖向好。1—3月份，农产品出口178.9亿美元，增速由去年的-0.1%转为9.2%。

【速抢】星系节点IPFS矿机新品上线 IPNodes、 IRacks系列【重磅开售】

2018年5月21日 下一个千亿级项目

IPNodes星系节点

作为2018年最具备投资价值的明星主链，星系节点IPFS矿机已经上线开售了，我们都知道，第一批挖比特币、以太坊的矿工都赚了很多，据媒体报道，首批挖矿的比特币矿工平均回报率是五倍，而以太坊的平均回报率是三倍，由此可见“第一批”挖矿的重要性，而IPFS是被称为和以太坊同级别的明星级主链，首批的挖矿价值，不言而喻。

由于星系节点IPFS矿机需要贡献存储空间、上下行带宽来获得奖励，因此IPFS的矿机并不过于依赖硬件，在电量消耗等问题上也要“亲民”很多。所以对于矿工来说，初期的投入成本很低。不过正因如此，所以市面上涌现了很多所谓的“IPFS矿机”公司，打着挖矿的名义在大规模进行虚假营销，在IPNodes星系节点这里，希望可以帮助到各位挑选矿机。

IPFS & Filecoin介绍

IPFS（Filecoin） 众筹 的情况

IPFS（Filecoin）众筹于2017年9月结束，以仅仅出售10%的份额就筹集到了2.57亿美元，打破了当时的众筹融资额记录。当时的硬顶是2亿代币（10%），最后其成交价格大约为2.5USD/Filecoin。

这意味着IPFS（Filecoin）还没正式上线，市值就已经达到了25亿美金。IPFS（Filecoin）被称为下一代网络协议，极有可能会成为一个千亿美金级别的项目。

IPFS（Filecoin）将分配给4种主要参与在Filecoin网络中的群体，代币分配如下：

70% 分配给Filecoin矿工（挖矿奖励-线性释放）

15%分配给Protocol实验室（原始分配）

10%分配给投资者（众筹公售）

5%分配给Filecoin基金会

IPFS（InterPlanetary File System，星际文件系统）是永久的、去中心化保存和共享文件的方法，这是一种内容可寻址、版本化、点对点超媒体的分布式协议。

内容可寻址：通过文件内容生成唯一哈希值来标识文件，而不是通过文件保存位置来标识。相同内容的文件在系统中只会存在一份，节约存储空间

版本化：可追溯文件修改历史

点对点超媒体：P2P 保存各种各样类型的数据可以把 IPFS 想象成所有文件数据是在同一个 BitTorrent 群并且通过同一个 Git 仓库存取。它集一些成功系统（分布式哈希表、BitTorrent、Git、自认证文件系统）的优势于一身，是一套分布式、P2P的文件存取系统。

“IPFS可以通过一种类似于网络将所有网站连接在一起的方式来连接所有这些不同的区块链。你可以在一个网页上添加一个与另一个页面相连的链接，这种方式例如同样可以在以太坊中添加一个与Zcash相连的链接，只要通过IPFS就能实现这一点。

从2G、3G到4G，从互联网到移动互联网，每次互联网底层技术的跃迁，都对现实的商业世界和生活引起巨变。IPFS这样的项目，能够从很底层的通信方式和网络结构上来颠覆和改变我们现有的网络基础。根据IPFS的设想，IPFS主要解决两个问题。第一是数据传输方式，HTTP的中心化传输方式将变成BT的点对点传输方式。第二是数据存储方式，由原来的原生数据存储方式变为Merkle DAG结构存储方式。 

而Filecoin天生自带巨大的分布式存储，与IPFS之间相互激励。IPFS使用的更多，对Filecoin的需求更大，存储需求更大，Filecoin的矿工节点便越多，对IPFS的支撑力度就越大，整个网络会变得更好用。

IPFS使用基于内容的寻址替代基于域名的寻址，将以前中心化（HTTP）的数据传输方式改进为点对点的bt传输方式。理论上可以使整个传输网络的速度更快、更安全、更健壮和更高效。

董天一表示，IPFS的核心是一个BT软件。而BT软件本身就自带激励机制，在IPFS上，这种激励机制叫做信用机制。“它实际上是BT软件‘人人为我，我为人人’的概念，就是说你乐于分享数据，别人也乐意把数据分享给你，这就是信任机制。”

大家都说的IPFS挖矿其实是不准确的。Filecoin体系内的交易市场和检索市场并不是挖矿，而是已有币之间进行的交易，实际上更应该理解为是一个自动撮合的市场，单纯的交易。就好比Filecoin自带了一个价值市场。

Filecoin矿机与传统的比特币矿机、以太坊矿机完全不同，我们不应该总是用老眼光看待新事物。从算力概念上来讲，传统的POW矿机生产出来就可以知道它的算力大小，收益曲线也很容易推算出来。 但是Filecoin矿机即使摆在面前，也无法得知其具体算力，因此也就无法推测回本周期。唯一的办法就是等待Filecoin网络上线之后，有了全网总算力，用户将自己的矿机接入网络，下载数据后，才有自身的算力，也只有到了这时，才产生真正的“算力”的概念。

星系节点作为专业的IPFS矿机销售托管解决方案专家，一直在IPFS中国地区传播和布道IPFS的核心理念。同时也在布局IPFS生态系统。矿机只是一个开端。对于IPFS矿机的布局上，我们远远走在同行的前面。我们矿机已经开始量产。

人社部：我国就业人员超7.7亿人 农民工总量逾2.8亿人

2018-05-22 13:33:00

人民网北京5月22日电 据人力资源和社会保障部网站消息，人社部昨日发布《2017年度人力资源和社会保障事业发展统计公报》。

IPFS & Filecoin介绍

IPFS（Filecoin） 众筹 的情况

IPFS（Filecoin）众筹于2017年9月结束，以仅仅出售10%的份额就筹集到了2.57亿美元，打破了当时的众筹融资额记录。当时的硬顶是2亿代币（10%），最后其成交价格大约为2.5USD/Filecoin。

IPFS（Filecoin）将分配给4种主要参与在Filecoin网络中的群体，代币分配如下：

70% 分配给Filecoin矿工（挖矿奖励-线性释放）

15%分配给Protocol实验室（原始分配）

10%分配给投资者（众筹公售）

5%分配给Filecoin基金会

IPFS（InterPlanetary File System，星际文件系统）是永久的、去中心化保存和共享文件的方法，这是一种内容可寻址、版本化、点对点超媒体的分布式协议。

内容可寻址：通过文件内容生成唯一哈希值来标识文件，而不是通过文件保存位置来标识。相同内容的文件在系统中只会存在一份，节约存储空间

版本化：可追溯文件修改历史

点对点超媒体：P2P 保存各种各样类型的数据可以把 IPFS 想象成所有文件数据是在同一个 BitTorrent 群并且通过同一个 Git 仓库存取。它集一些成功系统（分布式哈希表、BitTorrent、Git、自认证文件系统）的优势于一身，是一套分布式、P2P的文件存取系统。

“IPFS可以通过一种类似于网络将所有网站连接在一起的方式来连接所有这些不同的区块链。你可以在一个网页上添加一个与另一个页面相连的链接，这种方式例如同样可以在以太坊中添加一个与Zcash相连的链接，只要通过IPFS就能实现这一点。

从2G、3G到4G，从互联网到移动互联网，每次互联网底层技术的跃迁，都对现实的商业世界和生活引起巨变。IPFS这样的项目，能够从很底层的通信方式和网络结构上来颠覆和改变我们现有的网络基础。根据IPFS的设想，IPFS主要解决两个问题。第一是数据传输方式，HTTP的中心化传输方式将变成BT的点对点传输方式。第二是数据存储方式，由原来的原生数据存储方式变为Merkle DAG结构存储方式。 

而Filecoin天生自带巨大的分布式存储，与IPFS之间相互激励。IPFS使用的更多，对Filecoin的需求更大，存储需求更大，Filecoin的矿工节点便越多，对IPFS的支撑力度就越大，整个网络会变得更好用。

IPFS使用基于内容的寻址替代基于域名的寻址，将以前中心化（HTTP）的数据传输方式改进为点对点的bt传输方式。理论上可以使整个传输网络的速度更快、更安全、更健壮和更高效。

董天一表示，IPFS的核心是一个BT软件。而BT软件本身就自带激励机制，在IPFS上，这种激励机制叫做信用机制。“它实际上是BT软件‘人人为我，我为人人’的概念，就是说你乐于分享数据，别人也乐意把数据分享给你，这就是信任机制。”

住房需求持续释放市场预期较高 4月北京房租价格微涨

2018-05-22 09:18:00

　　4月全市房租价格微涨

　　本报讯(记者 孙杰)4月本市居民消费价格总水平(CPI)同比上涨2.5%，与上月持平，高于全国平均水平0.7个百分点。预计5月整体价格水平(CPI)同比上涨将持续在2%左右运行。房租方面，住房需求持续释放，房租价格环比上涨0.5%，同比上涨3.5%。昨天，市发改委价格监测中心发布4月本市重要商品价格运行分析及5月价格走势预测，公布了如上数据。

　　数据显示，4月本市CPI食品类价格同比上涨2.2%，环比下降1.7%。随着气温回升，北方蔬菜逐渐进入供京高峰期，影响鲜菜价格季节性回落；生猪产能供大于求，叠加消费需求淡季，价格持续回落至近8年最低水平；鸡蛋市场供应充足，环比下降3%。市价格监测中心的监测数据显示，25种蔬菜平均批发价为1.25元/500克；猪肉6.42元/500克；鸡蛋3.52元/500克。在京津冀农产品价格对比中，本市价格居三地的中位水平，其中猪肉价格最低，分别低于天津市和河北省12.34%和38.6%。

　　4月，CPI中服务项目价格环比上涨1.4%，同比上涨4.2%。清明节和劳动节假期出行需求增加，飞机票、旅行社收费、旅馆住宿价格分别上涨，景区实施旺季票价，多重因素拉动旅游服务类价格环比上涨5.6%。监测还显示，企业用工成本增加，导致劳动密集型行业价格上涨。服务业普通员工平均收入同比上涨19.02%，其中，家政服务业和快递业分别增长13.4%和11%。

　　在房租方面，4月CPI中房租价格环比上涨0.5%，同比上涨3.5%。这主要是因为，自去年起本市加强规范房屋租赁市场，住房需求持续释放，市场预期保持较高水平，4月房租价格持续小幅上涨。

　　5月重要商品价格走势预测农产品价格将继续下降。据相关负责人介绍，4月随着本市蔬菜产地轮换顺利完成，周边产区蔬菜大量上市，蔬菜价格将继续下降。而天气渐热，肉类消费需求进入季节性淡季，猪肉价格持续低迷，鸡蛋市场供应充足，加之玉米饲料价格持续走低，预计未来一段时间内本市农产品价格将进入季节性下降区间。

　　5月的房租价格也将以稳为主。“随着住房调控政策取向稳定，住房市场成交明显降温，住宅租赁市场供求形势宽松，加之进入传统租房淡季，预计接下来房屋租赁市场将平稳运行。”这位负责人说。

　　此外，受天气、供需等因素影响，国际农产品期货价格呈普涨态势，后期受南美产区异常天气和对美元汇率大幅走低因素影响，预计未来一段时间国际市场农产品价格将继续小幅上涨。由于全球原油库存不断下降，成品油需求预期持续向好，预计短期国内成品油价格将继续上调。
